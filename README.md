📚 Offre Stage – Application Java Swing

Application desktop développée en Java avec Swing permettant de gérer des offres de stage, des étudiants, des entreprises et un administrateur.

Le projet utilise JDBC pour se connecter à une base de données MySQL et propose une interface graphique permettant aux différents utilisateurs d’interagir avec les données.

Fonctionnalités

L’application permet :

👨‍🎓 Étudiants

Création d’un compte étudiant

Connexion à l’application

Consultation des offres de stage

Candidature à une offre

🏢 Entreprises

Création d’un compte entreprise

Publication d’offres de stage

Gestion des offres proposées

👨‍💼 Administrateur

Connexion administrateur

Gestion des offres

Validation des offres publiées

🖥️ Technologies utilisées

Java

Java Swing (interface graphique)

JDBC

MySQL

MySQL Workbench
